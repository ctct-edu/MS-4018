# 演習：Copilot in PowerPointを使用してプレゼンテーションを作成する

> 所要時間：20分

---

## 想定シナリオ

あなたは Contoso 社のマーケティングマネージャーです。
 現在、チームではラテンアメリカ市場向けに投入予定の新しいチャイティー製品ラインについて、クライアントに戦略を説明するための重要な打ち合わせを控えています。

この打ち合わせで使用するプレゼンテーションでは、次の点を満たす必要があります

- 製品の特徴・利点・ユースケースを、マーケティング視点で分かりやすく伝えること
- Contoso 社のブランド基準に沿った表現やデザインを維持すること
- すでに作成されているビジネスプラン（Word 文書）の内容を有効に活用すること

プレゼンテーションの主な対象者は、製品導入や販売戦略の意思決定に関わるクライアントの担当者であり、限られた時間の中で、要点を簡潔かつ魅力的に伝えることが求められます。
プレゼンテーションの作成後は、Copilot を活用しながらチームメンバーと共同編集を行い、最終的にマネージャーのレビューを受けて完成形に仕上げる予定です。

この演習を終える頃には、以下のことができるようになります

- Word 文書を元に新しいプレゼンテーションを作成すること  
- 画像を挿入すること  
- テキストを編集すること  
- プレゼンテーションを整理すること  

このような状況を想定して、Copilot in PowerPointを活用し、既存資料をもとにしたプレゼンテーションの作成、編集、改善の一連の流れを体験していきます。

**補足**：右側のCopilotペインの入力欄に **編集**もしくは**チャット**という項目があります。 　365Copilotには2つのモードがあります。

-  **編集を許可する** 

  「Copilot はドキュメントを直接編集できます。」

-  **チャットのみ** 

  「Copilot はチャットでの応答のみ」

---

### 1. Word 文書を元に新しいプレゼンテーションを作成する

1. PowerPoint アプリを開き、**新しいプレゼンテーションの作成**を開始します。

2. リボンの**ホームタブ**にある**Copilotアイコン**をクリックして、Copilot ペインを開きます。

3. 入力欄にある**＋（作業コンテンツを追加）ボタン**をクリックし、検索バーに **Promotion Plan for Chai Tea in Latin America.docx** と入力しファイルを選択する。そして、以下の内容を入力しスライドを作成します。

   ```prompt
   添付のファイルに基づいて、Contosoのチャイティーに関するプレゼンテーションを作成してください。
   ```

   ※スライド作成中、Copilot チャット内で追加の質問が表示されることがあります。必要に応じて回答してください。

4. この後の作業では、作成されたプレゼンテーション内容を改善するためにいくつかの作業を実施します。

---

### 2. アイスブレイク用スライドを作成する

1. 右側のCopilotペインの入力欄のモードを**チャットのみ**に変更します。

2. 以下のプロンプトを入力します。

   ```prompt
   グローバルな聴衆に向けて良いアイスブレイク用の質問を考えてください。3つのパターンを出力してください。
   ```

3. Copilot は質問案を提案します。もしどれも合わなければ、さらに案を出すよう依頼できます。

4. 入力欄のモードを**編集を許可する**に変更します。

5. 提案された内容の中で気に入ったものを選び、それを使ってスライドを作るように依頼します。以下のようにプロンプトを入力します（冒頭の部分を一部書き換えてください）。

   > 注：Copilotペインでスライドの作成ができないと回答された場合は、スライド左上のCopilotアイコンをクリックしてスライドもしくはセクションの追加からスライドの生成を依頼します。

   ```prompt
   [気に入った質問の番号もしくは内容]について聴衆へ問いかけ、一緒に考えるような内容のスライドを生成してください。
   ```

6. 新しい単体のスライドが作成されます。作成されたスライドは順番の入れ替えや編集することも可能です。

---

### 3. 画像を挿入する

Copilot はスライド生成時点で画像を含めることがありますが、より具体的なものにしたい場合は独自に画像を追加・編集できます。もし管理者が設定していれば、承認された企業ライブラリからの画像を使うこともできます。あるいは、アイデアに基づいて新しく画像を生成させることも可能です。 

1. タイトルスライドに移動してから、Copilotペインで以下のプロンプトを入力してください。

   ```prompt
   タイトルスライドの画像として使用するために、暖かいカップのチャイを飲んでいる人の画像を生成してください。
   ```

2. 画像が配置されている場合は新しい画像をスライドのスペースに最も合うように調整します。

---

### 4. テキストを編集する

プレゼンテーションを見返していると、Copilot が生成したテキストが必ずしもぴったりではないと感じることがあります。そんなときは Copilot にテキストの改善を依頼できます。たとえば「このスライドのテキストをもっと簡潔にして」「読み手にとってもっと魅力的な文にして」などの方法で行えます。

1. 文章が書かれている任意のスライドに移動します。  

2. 入力欄のモードを**編集を許可する**に変更します。

3. Copilotペインで以下のようにプロンプトを入力します。

   ```prompt
   今開いているスライドのテキストをもっとカジュアルな文体にしてください
   ```

4. Copilot が提案する改訂案をレビューし、必要に応じて修正します。

5. テキストボックスを選択することでもCopilotの機能を使用することが可能です。自動リライトによる書き換えなど、テキストに対して変更を加えられます。
